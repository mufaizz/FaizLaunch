## What is FaizLaunch?

FaizLaunch is a smart game installer and launcher built for gamers who use FitGirl Repacks, DODI, and other compressed game sources. It solves every problem those gamers face — failed installs, DLL errors, Windows Defender deleting files, 13-hour HDD installs that crash at 99%.

**FaizLaunch is not a game store. You bring your download link — FaizLaunch handles everything else.**

---

## The Problem We Solve

- Install fails at 99% after 13 hours — no resume, start over
- DLL errors with cryptic messages — no explanation, no fix
- Windows Defender deletes essential files mid-install
- Fake repack sites with crypto miners and malware
- No way to know if your PC can run the game before downloading 50GB
- Corrupted downloads with no way to verify

## Installation

1. Go to [Releases](https://github.com/mufaizz/FaizLaunch/releases)
2. Download `FaizLaunch-Setup-v1.0.0.exe`
3. Run as Administrator
4. Windows may show a warning — click **More info** → **Run anyway**

## Build from Source
```bash
git clone https://github.com/mufaizz/FaizLaunch.git
cd FaizLaunch
npm install --legacy-peer-deps
npm run dev
```

## FAQ

**Is this piracy?**
No. FaizLaunch does not provide or distribute games. It is a tool — like 7-Zip or IDM.

**Does it work offline?**
Yes. 100% offline after installation. No accounts, no servers, no tracking.

**Why does Windows warn about it?**
FaizLaunch is not code signed. Full source is public — verify it yourself.
